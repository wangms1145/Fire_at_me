还在开发，别急

别守着那个release页面不放了，发布之前会隐藏这个项目

## 安装
_你需要**2022.3.47f1**版本的Unity编辑器才能玩这个游戏._

则么才能抢鲜体验?
1. 下载Unity Hub和编辑器
2. 把这个项目复制到一个你找得到的位置
3. 点击unity hub中的`添加项目`然后选择文件夹`fire_at_me`
